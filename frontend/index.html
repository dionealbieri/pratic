💾 Fazer Backup

Gera um arquivo .db com todos os dados do sistema. Guarde em local seguro — armazena clientes, pedidos, produção, estoque e configurações.

💡 Recomendado: faça backup diariamente ou antes de grandes atualizações.
📂 Restaurar Backup

Restaura o banco de dados a partir de um arquivo .db de backup. Atenção: os dados atuais serão substituídos.

📁
Arraste o arquivo .db ou clique para selecionar
🗑️ Limpar Dados

Remove dados operacionais mantendo cadastros. Útil para iniciar um novo período ou limpar dados de teste.

📋 Boas práticas
✅ Faça backup antes de atualizar o sistema
✅ Guarde os backups em pasta diferente do sistema
✅ Nomeie os backups com a data para controle
✅ Teste a restauração periodicamente
⚠️ Após restaurar, reinicie o servidor
⚠️ A restauração substitui todos os dados atuais

❓ Ajuda - Painel do Gestor

Este painel centraliza as principais métricas de tomada de decisão e PCP. Abaixo, veja como utilizar cada função:

📊 Interpretação do Dashboard:
  • Aderência à Meta: Porcentagem de dias no mês em que a meta diária foi atingida.
  • Índice de Perdas (%): Relação entre peças perdidas e a produção total acumulada.
  • Custo de Perdas (R$): Custo financeiro estimado das perdas do mês, calculado com base no custo de compra do produto.
  • Insumos Críticos: Produtos do estoque que estão com saldo abaixo ou igual ao estoque mínimo.
🏭 Como Registrar Produção Diária:
  1. Clique no botão + Registrar Produção no topo direito da tela de Lançamento.
  2. Informe a Data, o Colaborador, a Máquina e a meta padrão.
  3. (Opcional) Vincule a um Pedido de venda ativo ou insira um número manual.
  4. Em Produtos Produzidos, use o botão laranja + Adicionar Produto para escolher o item e registrar a quantidade de peças produzidas, perdas (especificando tipo) ou sobras.
  5. Confirme clicando em Salvar.
⚙️ Filtros e Navegação:
  • Utilize o seletor de Mês no topo para recarregar todos os dados do sistema correspondentes àquele período.
  • Utilize os menus laterais para navegar entre os módulos de Vendas, PCP, Segurança, EPIs e Configurações Gerais.
